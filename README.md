# Rara App 🛍️

Aplicación web tipo tienda online desarrollada con Django, con panel de administración personalizado para gestionar productos y categorías.

---

## 🚀 Funcionalidades

### 🛒 Tienda
- Catálogo de categorías con imágenes
- Listado de productos por categoría
- Estado de productos: disponible / agotado (sin eliminar)
- Diseño responsivo con cards

### 🧑‍💻 Panel de administración
- Dashboard con métricas básicas
- Crear y gestionar productos
- Crear categorías
- Activar/desactivar productos

---

## 🧱 Tecnologías
- Python / Django
- HTML, CSS, JavaScript
- Font Awesome

---

## 📁 Estructura básica

- `/` → Tienda principal  
- `/categoria/<slug>/` → Productos por categoría  
- `/panel/` → Dashboard  
- `/panel/productos/` → Gestión de productos  
- `/panel/categorias/crear/` → Crear categorías
